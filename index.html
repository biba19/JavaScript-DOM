Interaction avec le DOM

Qu'est-ce que le DOM ?

Le Document Object Model (DOM) est une interface de programmation d'applications (API) permettant de manipuler des documents HTML. Le DOM représente un document HTML comme un arbre de nœuds. Le DOM fournit des fonctions qui vous permettent d'ajouter, de supprimer et de modifier efficacement des parties du document. Notez que le DOM est un moyen de manipulation des documents HTML et XML indépendant de la plate-forme et du langage. What is Document Object Model (DOM)
